# Projects Dashboard

## Project workflow

- **Active:** work with a defined outcome and a current next action
- **Someday:** worthwhile ideas that are not currently committed
- **Completed:** finished projects kept for reference

## Current project folders

- Opalitemakers
- StreamPulse
- Trading Journal

These existing folders have not been changed. When you are ready, create an overview note inside each one using the [[10 Templates/Project|Project template]].

## Review prompts

- What is the single next physical action for each active project?
- Is anything active without a clear outcome?
- Should anything move to Someday or Completed?
- Did a project produce knowledge that belongs in a permanent reference note?

## Create a project

1. Create a note in `02 Projects/Active`.
2. Insert the [[10 Templates/Project|Project template]].
3. Define the outcome and next action.
4. Link supporting notes instead of duplicating them.
