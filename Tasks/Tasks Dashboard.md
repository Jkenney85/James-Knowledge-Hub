# Tasks Dashboard

Each task is a full note with room for context, subtasks, decisions, links, and completion notes.

## Create a task

1. Create a new note inside `Tasks/Task Notes`.
2. Give it a short, action-oriented title such as `Schedule Mustang alignment`.
3. Insert the [[10 Templates/Task|Task template]].
4. Set its status, priority, area, project, and due date.

Use `YYYY-MM-DD` for due dates. Leave `due` empty if the task has no real deadline.

## Task views

Use the view selector in the Base below to switch among:

- **Open tasks:** everything still actionable
- **Overdue:** unfinished tasks with a past due date
- **Due today:** tasks due today
- **Upcoming:** tasks due during the next 14 days
- **High priority:** urgent and high-priority work
- **Unscheduled:** open tasks without a due date
- **Waiting:** tasks blocked by another person or event
- **Completed:** finished task notes

## Working conventions

- `open` — ready to work
- `in-progress` — currently being worked
- `waiting` — blocked or delegated
- `completed` — finished
- `cancelled` — intentionally abandoned

Use priorities sparingly: `urgent`, `high`, `medium`, or `low`. A due date means something is genuinely time-sensitive—not merely that it would be nice to finish soon.
